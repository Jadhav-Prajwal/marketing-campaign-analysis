# Marketing Campaign Performance Dashboard (Excel)

## Project Overview

This project focuses on analyzing marketing campaign performance using Microsoft Excel to uncover actionable business insights. The analysis evaluates how different marketing channels, campaign types, and customer segments influence key performance metrics such as engagement, traffic generation, conversion rates, and return on investment (ROI).

The workflow includes data cleaning, KPI development, exploratory analysis using pivot tables, and the creation of an interactive dashboard for visualizing campaign performance.


## Dataset
Dataset: **Marketing Campaign Performance Dataset**

Source: **Kaggle**

The dataset includes marketing campaign data such as:
- Campaign Type
- Marketing Channel
- Customer Segment
- Impressions
- Clicks
- Conversion Rate
- ROI
- Engagement Score
- Campaign Dates



## Tools and Techniques
- Microsoft Excel
- Data Cleaning and Preparation
- KPI Calculations
- Pivot Tables and Pivot Charts
- Dashboard Design



## Key Performance Indicators (KPIs)
The following KPIs were calculated and analyzed:

- Click Through Rate (CTR)
- Conversion Rate
- Return on Investment (ROI)
- Engagement Rate
- Engagement Efficiency
- Reach Score
- Conversion per Click


## Dashboard Features
The interactive Excel dashboard includes:

- ROI by Marketing Channel
- Engagement by Customer Segment
- Click Distribution by Marketing Channel
- Conversion Rate by Campaign Type
- Monthly Campaign Performance Trend
- KPI summary cards for quick performance evaluation



## Key Insights
- Email marketing generated the highest ROI and overall traffic.
- Influencer campaigns achieved the highest conversion rates.
- The Foodies customer segment showed the highest engagement levels.
- Campaign performance remained relatively consistent across months.

## Business Recommendations
- Increase investment in email marketing campaigns due to strong ROI performance.
- Expand influencer marketing strategies to improve campaign conversions.
- Develop targeted campaigns for highly engaged customer segments.
- Optimize underperforming marketing channels through improved targeting and creative strategies.
